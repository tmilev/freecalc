\begin{definition}[Complex numbers]The set of complex numbers $\mathbb C$ is defined as the set
\[
\{a+  b i  | a,b-\text{real~numbers}\},
\]
where the number $i$ is a number for which 
\[
\alertNoH{2,12}{i^2=-1}\quad .
\]
The number $i$ is called the imaginary unit. \uncover<3->{\alertNoH{3,4}{By definition, }}\uncover<2->{$\uncover<handout:0|2,3>{\alertNoH{2,3}{\pm}} \alertNoH{2}{\sqrt{-1}= i}$}.
\end{definition}
