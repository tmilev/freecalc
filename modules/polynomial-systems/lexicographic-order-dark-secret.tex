\begin{frame}
\begin{observation}[The dark secret of the lexicographic order]
The lexicographic order is not the dictionary order ...

\uncover<6->{\alertNoH{6}{... unless we regard monomials as fixed-length words written using the infinite alphabet $0,1,2,\dots$ given by the variable exponents.}}
\end{observation}

\begin{itemize}
\item<2-> In the lexicographic order: 
$
\begin{array}{rcrcrcr}
\alertNoH{4}{x^{2}}\alertNoH{5}{y} &>& \alertNoH{4}{x}\alertNoH{5}{y} &>& \alertNoH{4}{x} &>&\alertNoH{5}{y^2} \\
\alertNoH{4}{xx}\alertNoH{5}{y} &>& \alertNoH{4}{x}\alertNoH{5}{y}& > & \alertNoH{4}{x} &>& \alertNoH{5}{yy}\\
\uncover<4->{
\alertNoH{4}{2},\alertNoH{5}{1} &>& \alertNoH{4}{1},\alertNoH{5}{1} & > & \alertNoH{4}{1},\alertNoH{5}{0} &>&\alertNoH{4}{0},\alertNoH{5}{2}}
\end{array}
$.

\item<3->The second row is not an order one sees in a dictionary, in either direction.

\item<6-> \alertNoH{6}{The third row is the order one sees in a dictionary, if using two-letter words with an infinite alphabet with letters: $0,1,2,\dots$.}
\end{itemize}
\end{frame}
