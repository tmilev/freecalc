\begin{frame}
We can generalize the previous example to other powers of $x$:
\begin{theorem}[Infinite Limits of $\frac{1}{x^r}$]
If $r > 0$ is a rational number, then
\[
\lim_{x\to\infty} \frac{1}{x^r} = 0.
\]
If $r > 0$ is an integer
\[
\lim_{x\to - \infty} \frac{1}{x^r} = 0.
\]
\end{theorem}
\end{frame}
