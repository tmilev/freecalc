Compute the composite functions $(f\circ g)(x)$, $(g\circ f)(x)$. Simplify your answer to a single fraction. Find the domain of the composite function. The answer key has not been fully proofread, use with caution.

g{}({{x}})=\frac{x-1}{x+2}$.

\begin{tabular}{rl}

\end{tabular}
}
