\begin{definition}[Infinitely Smooth Variable Change]
An infinitely differentiable map $\fcv f:\mathbb R^n\to \mathbb R^n$ is called an (infinitely) smooth variable change. 
\end{definition}
\begin{itemize}
\item<2-> Variable change $\fcv f:\mathbb R^n\to \mathbb R^n$ is given by f-ns $f_1,\dots, f_n$. \uncover<3->{We write:

\hfil $
\fcv f:\left|\begin{array}{r@{~}c@{~}l}
\uncover<5->{\alert<5>{x_1} &=&} f_1(\alert<4>{ y_1, \dots, y_n})\\
&\vdots&\\
\uncover<5->{\alert<5>{x_n} &=&}f_n(\alert<4>{ y_1, \dots, y_n}) \quad .
\end{array}\right.
$
}
\item<4-> The variables $\alert<4>{y_1,\dots, y_n}$ denote coordinates in the domain of $\fcv f$.
\item<5-> We may include vars. $\alert<5>{x_1,\dots, x_n}$ denoting coords. in codomain of $\fcv f$.
\item<6-> Fix $\alert<6>{y_2 ,\dots, y_n}$ and view $\fcv f$ as  curve with respect to $y_1$; plot.
\item<11-14,15-> Do similarly with respect to the remaining variables.
\end{itemize}
